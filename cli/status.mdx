---
title: "lazy status"
description: "List all your synced configs."
---

## Usage

```bash
lazy status
```

Lists all configs saved in your account with their name, tool, version number, and last updated timestamp.

## Example output

```
Name        Tool     Version   Updated
default     claude   3         2 hours ago
work        claude   1         3 days ago
default     cursor   2         1 week ago
```
