## Request

```bash
GET /api/teams/:slug
```

Returns team details and members. Requires team membership.

### Path parameters

| Parameter | Type | Description |
|---|---|---|
| `slug` | `string` | Team slug |

### Example

```bash
curl https://lazyraccoon.dev/api/teams/my-team \
  -H "Authorization: Bearer lr_your_key"
```
